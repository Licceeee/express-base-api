
# Base-API

clone
```https://github.com/Licceeee/express-base-api.git```

cd into dir
```cd express-base-api```

install dependencies
```npm i```

create .env and add credentials
```cp .env.sample .env; vi .env```

replace api-domain.com with real domain link in index.html

run with nodemon
```npm run dev```

```http://localhost:3030/```

or 

```http://localhost:<PORT>/```
